# loading-utils

A canvas-based editor for creating custom loading animations. Built with Nuxt 4, Vue 3, and Konva.

## Installation

**Prerequisites:** Node.js 18+ and [bun](https://bun.sh)

```bash
bun install
```

## Development

```bash
bun run dev
```

Visit `http://localhost:3000` to view the application.

## Build

```bash
bun run build
```
